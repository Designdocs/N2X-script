# N2X
A V2board node server based on Xray-Core.

一个基于Xray的V2board节点服务端，支持V2ay,Trojan,Shadowsocks协议

Find the source code here: [InazumaV/N2X](https://github.com/Designdocs/N2X)

如对脚本不放心，可使用此沙箱先测一遍再使用：https://killercoda.com/playgrounds/scenario/ubuntu

# 详细使用教程

[教程](https://n2x.v-50.me/)

# 一键安装

```
wget -N https://raw.githubusercontent.com/Designdocs/N2X-script/master/install.sh && bash install.sh
```
